# 奇门五维取象 Playbook

## 统一映射

- 宫位/宫五行：优先回答 `性` 与 `体`
- 八门：优先回答 `情` 与行动方式
- 九星：优先回答 `形` 与外显资质
- 八神：优先回答隐性 `情` 与潜在动机
- 格局/季节旺衰/空亡：优先回答 `用` 与节奏

## 重写模板

```markdown
### 五维落点
- 性：这个宫位在当前问题中的根本气质是什么
- 情：门神让它呈现出什么状态
- 形：九星让它显得像什么
- 体：在此题场景中究竟对应谁/何事/何地
- 用：它到底推动、阻碍、掩护、延迟还是转机
- 时空坐标：当前季节、阶段、场域是什么
```

## 约束

- 不允许只报门星神名词，不解释其在当前问题中的作用。
- 不允许只说吉凶，不说它如何落到当前问题对象上。
- 不允许把门、星、神的象意孤立使用，必须回到用神落宫主线。
